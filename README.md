# Winners Baptist Church Website

A modern, responsive website for Winners Baptist Church built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Mobile-friendly layout that works on all devices
- **Navigation Menu** - Hamburger menu for mobile devices
- **Smooth Scroll Effects** - Reveal animations as you scroll
- **Header Effects** - Dynamic header that responds to scroll position
- **Event Preview** - Display upcoming church events
- **Giving Section** - Information about supporting the ministry

## Pages

- **Home** (`index.html`) - Main landing page
- **About** (`about.html`) - Church information
- **Live** (`live.html`) - Live streaming or services
- **Events** (`events.html`) - Upcoming events
- **Contact** (`contact.html`) - Contact information
- **Give** (`give.html`) - Donation and support information

## Getting Started

1. Clone or download the project
2. Open `index.html` in your web browser
3. Navigate through the different pages using the menu

## Technologies Used

- **HTML5** - Structure and markup
- **CSS3** - Styling and responsive design
- **JavaScript** - Interactive features and animations

## Features Explained

### Mobile Menu
The hamburger menu automatically appears on smaller screens (≤768px width) to provide better navigation.

### Scroll Reveal
Sections fade in as you scroll down the page, creating an engaging user experience.

### Responsive Header
The header adjusts its styling when the page is scrolled down, creating a sticky navigation effect.
